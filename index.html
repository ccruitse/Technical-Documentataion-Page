Introduction

Website development is the process of building websites and applications. It focuses more on coding and making sure a website functions well. It’s essentially the usability aspect of websites and apps.

The following topics will be discussed, to further understand website development.

  • What is Front end development?
  • What is Back end development?
  • What is the Difference of Front end and Back end development?
  • What is Full stack development?
  • What are the Program Languages used for development?
  • Sample projects.
Front end development

Front end development is a style of computer programming that focuses on the coding and creation of elements and features of a website that will then be seen by the user.

Some common programming languages used for Front end development:

  • HTML
  • CSS
  • Javascript

Front end development is programming which focuses on the visual elements of a website or app that a user will interact with (the client side).

Front end also works in its own set of frameworks and libraries. Here are just a few of the frameworks and libraries a front end developer would work with:

  • AngularJS
  • React.js
  • jQuery
  • Sass
Back end development

Back end development focuses on the side of the website users can’t see. It’s what makes a site interactive.

Some common programming languages used for Back end development:

  • Python
  • PHP
  • Java

Back end development focuses on the side of a website users can’t see (the server side).

Back end frameworks include:

  • Express
  • Django
  • Rails
  • Laravel
  • Spring
Full stack development
It refers to the development of both front end(client side) and back end(server side) portions of web application.
HTML
HTML stands for Hyper Text Markup Language. It's the standard markup language for creating webpages.
Elements

Lists of Elements

  • Anchor. Link to page or the section in the page. <a href"https:"></a> <a href"#feature"></a>
  • Article. Represent a self-contained composition in a document, page, application, or site. <article></article>
  • Aside. Represent a section of a page that consist of contents of content that is tangentially related to the content around the aside element and which could be considered separate from that content. <aside></aside>
  • Blockquote. Used to display the line quotation (a section that is quote from another source). <blockquote></blockquote>
  • Body. <body></body>
  • Break. Produces a line break in text (carriage-return). <br/>
  • Button. Clickable button. <button></button>
  • Caption. Describe what the table is about. <caption></caption>
  • Code. <code></code>
  • Commenting. Allows you to leave messages without affecting the browser display. <!-- -->
  • Declaration. Declaring HTML file. <!DOCTYPE html>
  • Division. Defines division or a section in HTML document. <div></div>
  • Emphasis. Emphasize the word. <em></em>
  • Fieldset. Used to group related inputs and labels together in a web form. <fieldset></fieldset>
  • Figure. Represents self-contained content and will allow you to associate an image with a caption. <figure></figure>
  • Figure Caption. Add a caption to describe the image contained within the figure. <figcaption></figcaption>
  • Footer. Container for a collection of content that is related to the page. <footer></footer>
  • Form. Web form. <form></form>
  • Gallery. <gallery></gallery>
  • Head. <head></head>
  • Heading. <h1></h1> <h2></h2>
  • Header. Introduce the page <header></header>
  • Horizontal Rule. Represent a thematic break between paragraph- level element. <hr>
  • HTML. Set HTML file format. <html lang=""></html>
  • I. Represents a span of text in an alternate voice or mood, or otherwise offset from the normal prose in a manner indicating a different quality of text. <i>
  • Idiomatic text. Used for idiomatic text, or text that is separate from the "normal" text content. <i></i>
  • Image. Contains image source and alt attribute. <img src="" alt=""/>
  • Input. Allows you several ways to collect data from a web form. <input/>
  • Label. Used to help associate the text for an input element. <label></label> <label for=""></label>
  • Legend. Acts as a caption for the content in the fieldset element. <legend></legend>
  • Link. Link to styles.css or other page. may also import font family with font weight. <link rel="stylesheet" href="styles.css"/> <link rel="stylesheet" href="http://"/> <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800"/>
  • List Items. Create items in a list <li></li>
  • Main. Identify the main section of this page, contain the core content of your page. <main></main>
  • Meta. Set global setting, mobile scale and page descriptions. <meta charset="UTF-8"/> <meta name="viewport" content="width=device-width, initial-scale=1.0"/> <meta name="description" content="Quiz"/>
  • Navigate. Represents a section of a page that links to other pages or to parts within the page. Can navigate to the elemnet id. <nav></nav> <nav>href="#id"</nav>
  • Option. <option></option>
  • Ordered list. Numbered when displayed. <ol></ol>
  • Paragraph. Used to create a paragraph of text on websites. <p></p>
  • Section. Separate content into sections. <section></section>
  • Select. Dropdown type. <select></select>
  • Span. An inline container used to mark up a part of a text, or a part of a document. A generic inline container for phrasing content, which does not inherently represent anything. <span></span>
  • Strong. Used to indicate that some text is of strong importance or urgent. <strong></strong>
  • Style. Used to create a paragraph of text on websites. <style></style>
  • Table. <table></table>
  • Table Body. Which portion contains the primary data or content. <tbody><body;/td>
  • Table Data. Indicates a data cell. <td></td>
  • Table head. Are used to indicate which portion of your table is the header. <thead></thead>
  • Table header. Indicates a header cell. <th></th>
  • Table row. Used to indicate a table row. <tr></tr>
  • Text Area. Acts like an input element of type text. <textarea></textarea>
  • Title. Screen readers the content of the page. <title></title>
  • Unordered list. Bullet type. <ul></ul>
